# port-clean

Minimalist portfolio site for Aaron Blum. Built with React, TypeScript, and Vite.

## Stack

- React 19 + TypeScript + Vite
- React Router v7 (`createBrowserRouter`)
- GSAP + Framer Motion (animation pages)
- Pure CSS (single `styles.css`)
- Vitest + React Testing Library

## Pages

- `/` — Home (bio)
- `/cv` — Resume (experience, education, skills)
- `/work` — Projects with links
- `/animations` — SVG animation gallery
- `/tv` — Interactive retro television

## Commands

```
bun run dev       # dev server
bun run build     # production build
bun run preview   # preview production build
bun vitest run    # run tests
```
